Methodology

I used the pandas python library to explore the raw data.json provided by the challenge.

Ipython notebook showing data exploration

I then used the folium lib to generate a choropleth map to reveal how age and geography factor into the amount donated for the Sports and Environment categories.

Conclusions

Looking at the hotspots in the maps above, we can see that users aged 18-24 in Oregon and Colorado significantly out-donate the other age groups on those states. Therefore, this analysis suggests that BootLoader sends a push notification to all users aged 18-24 in Oregon and Colorado.

Limitations

The granularity of the map is also state-wide; looking at donation amounts by zip code or city could allow better targeting. It looks like Boulder, CO; Portland, OR; and Atlanta, GA are the actual hotspots.

My visualization only considers amount donated vs. age and geography. We could split the data into many different ways (by gender, device, geography, etc. and combinations of those).

Because I present one view of the data, my visualization does not enable a BootLoader employee to dynamically explore the data. A possible improvement could be: allow the user to select the split key (age, gender, device, marital_status), and then generate the choropleth maps based on their selection. I also produce n maps, where n is the number of possible values for any split key. Figuring out a representation that could reduce the n maps into one would be a huge improvement.
